# F1Paper

LaTeX project for an F1 technical paper (multiple chapter files).

## Quick start

Prerequisites:
- Install a TeX distribution (MacTeX on macOS).
- `latexmk` is recommended for easy builds.

Build PDF (recommended):

```bash
latexmk -pdf main.tex
```

Manual build (if not using `latexmk`):

```bash
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

If your setup uses `biber` instead of `bibtex`, replace the `bibtex` step accordingly.

## Project structure

- `main.tex` — master document
- `*.tex` — chapter/source files (e.g. `introduction.tex`, `aerodynamics.tex`, ...)
- `references.bib` — bibliography database
- `Figures Images/` — image assets used by the paper
- `Figures HTML/` — exported HTML versions of figures
- `f1_2026_submission/` — a submission-ready copy of the paper and assets

## Notes
- There are auxiliary files in the repo (`main.aux`, `main.bbl`, `main.toc`) generated by previous builds.
- Use `latexmk -c` to clean build artifacts.
